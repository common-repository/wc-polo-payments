== Description ==

Processar Pix nunca foi tão fácil! Com **configuração simples para todos**, comece a receber pagamentos na sua loja em poucos minutos.

Nossa plataforma usa tecnologia de ponta para fornecer um serviço seguro e eficiente, automatizamos os processos financeiros para você ampliar suas receitas.

= Repasses rápidos em tempo real =

Nosso sistema garante transferências instantâneas, receba o valor de suas vendas diretamente em sua conta em qualquer banco e em tempo real.

= Checkout Transparente =

Com o checkout transparente os seus clientes irão fazer o pagamento sem sair da loja através do QR Code ou do Copia e Cola.

= Aprovação Automática =

O status do pedido é atualizado automaticamente, eliminando a necessidade de comprovantes.

== Instalação ==

Faça o [cadastro na polopag](https://polopag.com/register/) e siga os passos abaixos:

1. No painel do Wordpress vá até a aba plugins e selecione "Adicionar Plugin"
2. Pesquise por "Polopag" e instale o plugin
3. Após ter instalado o plugin vá em WooCommerce > Configurações > Pagamentos
4. Ative e entre nas configurações do plugin Polopag
5. Nas configurações do plugin Polopag, adicione a API Key da sua conta Polopag. Caso deseje gerar uma nova API Key basta [clicar aqui](https://polopag.com/myapikey/)
6. Salve as alterações e o plugin ja estará funcionando!
